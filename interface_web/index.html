Projet Portail

Ceci est l'interface web pour gérer les utlisateurs pouvant ouvrir le portail grâce à leurs QR Code.

Les différentes fonctionnalités disponibles sont à gauche.



L'Inscription : permet d'inscrire un nouvel utlisateur pour pouvoir ensuite ouvrir le portail grâce au QR Code.

La Désinscription : permet de supprimer de la base de données d'un utilisateur, donc qui ne pourras plus ouvrir le portail avec son QR Code.

L'Envoi QR Code par mail : permet d'envoyer un mail avec le QR Code en pièce jointe si l'utilisateur est bien inscris.
